# 영업 대시보드

> Dataview 플러그인 필요 (설정 > 커뮤니티 플러그인 > Dataview 설치·활성화)

## 🔔 갱신 임박 (60일 이내)
```dataview
TABLE 다음갱신일 AS "갱신일", 유형태그 AS "태그", 연락처
FROM "고객DB"
WHERE type = "고객" AND 다음갱신일 AND date(다음갱신일) <= date(today) + dur(60 days)
SORT 다음갱신일 ASC
```

## 👶 자녀보험 미가입
```dataview
TABLE 가족구성 AS "가족", 연락처
FROM "고객DB"
WHERE type = "고객" AND contains(유형태그, "자녀보험미가입")
```

## 📋 전체 고객 현황
```dataview
TABLE 다음갱신일 AS "갱신일", 유형태그 AS "태그", 최근상담일 AS "최근상담"
FROM "고객DB"
WHERE type = "고객"
SORT 최근상담일 DESC
```
