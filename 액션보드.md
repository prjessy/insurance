# ✅ 액션 보드

> Dataview 플러그인 필요. 상담/처리 기록의 `다음 액션` 체크박스를 자동 집계합니다.
> (프로파일에 따라 폴더가 다름: 금융=상담기록 / 기술지원=처리이력 / 일반=기록)

## 🔴 미완료 액션 (전체)
```dataview
TASK
FROM "상담기록" OR "처리이력" OR "기록"
WHERE !completed
GROUP BY file.link
```

## 🟢 최근 완료
```dataview
TASK
FROM "상담기록" OR "처리이력" OR "기록"
WHERE completed
GROUP BY file.link
```

## 🗂 기록 현황 (최신순)
```dataview
TABLE 채널 AS "채널", status AS "상태", file.mtime AS "수정"
FROM "상담기록" OR "처리이력" OR "기록"
WHERE type
SORT file.mtime DESC
LIMIT 20
```

---
> 새 액션은 각 기록 노트의 `# 다음 액션` 아래 `- [ ] 할일` 형식으로 적으면 여기 자동 집계됩니다.
