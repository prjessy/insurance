# 개인정보 주의 체크리스트

이 볼트는 본인 PC에 로컬 저장됩니다. 보험사 사이트 자동 로그인/수집은 하지 않으며, 설계사 본인이 정당하게 확인한 정보만 정리합니다.

## 반드시 지킬 것
- [ ] **녹음 동의**: 상담 녹음 전 고객에게 동의를 받는다.
- [ ] **수집 최소화**: 영업에 필요한 정보만 입력한다 (주민번호 전체 등 민감정보 지양).
- [ ] **PC 보안**: 디스크 암호화(BitLocker) + 화면 잠금 설정.
- [ ] **클라우드 동기화 주의**: 볼트 폴더를 외부 클라우드와 동기화한다면 암호화 여부 확인.
- [ ] **Claude 입력 시**: 프롬프트에 붙여넣는 정보도 최소화. 불필요한 식별정보는 가린다.
- [ ] **보관·파기**: 더 이상 필요 없는 고객 정보는 주기적으로 파기한다.

> 정확한 의무사항은 소속 보험사/GA의 개인정보 처리방침과 개인정보보호법을 따르세요.
