# 보험 영업 자동화 볼트 🛡️

흩어진 고객 정보를 한곳에 모으고, **상담 녹음 → 정리 → 고객 분류 → 맞춤 제안서·문자**까지 Claude로 자동화하는 Obsidian 볼트입니다. 별도 API 비용 없이 Claude 앱(또는 Claude Code)에 프롬프트를 붙여 씁니다.

### 🌐 소개 사이트 (먼저 보세요)
**https://prjessy.github.io/insurance/** — 작동 방식·역할·시간 절약 효과를 예제와 함께 한눈에.

> ⚠️ 이 저장소는 **공개(PUBLIC)** 이며, 들어있는 고객·상품은 모두 **가상 데이터**입니다. 실제 고객 정보는 절대 공개 저장소에 올리지 마세요. 실사용 시 저장소를 Private으로 바꾸거나 로컬 볼트에만 두세요. → [개인정보-주의](개인정보-주의.md)

## 📁 구성

| 파일 / 폴더 | 용도 |
|------|------|
| `상품DB/` | 보험 상품·보장·조건 (한 번 정리해 재사용) |
| `고객DB/` | 고객별 가입 이력·갱신일·가족·유형 태그 |
| `상담기록/` | 상담 녹음 전사·정리 |
| `대시보드.md` | 갱신 임박·자녀보험 미가입 자동 집계 (Dataview) |
| `프롬프트.md` | 상담정리 / 제안서 / 타깃 메시지 프롬프트 |
| `시간절약-예제.md` | 항목별 AS-IS → TO-BE + 하루 절약 추정 |
| `개인정보-주의.md` | 동의·보안 체크리스트 |
| `index.html` | 소개 사이트(GitHub Pages) 소스 |

---

## 🚀 사용법

1. **Obsidian 설치** → 이 폴더를 "Open folder as vault"로 열기
   ```
   winget install Obsidian.Obsidian
   ```
2. 설정 → 커뮤니티 플러그인 → **Dataview** 설치·활성화 (대시보드 작동용)
3. `상품DB`·`고객DB`의 `_..._템플릿.md`를 복사해 새 노트 작성 (예시: `고객DB/정수남.md`)
4. 상담 후 → `프롬프트.md`의 **① 상담 정리**를 Claude에 붙여넣고, 결과를 고객 카드에 반영
5. `대시보드.md`에서 갱신 임박·자녀보험 미가입 고객 확인
6. `프롬프트.md`의 **② 제안서**, **③ 타깃 메시지**로 마무리

## 🔁 역할 (누가 무엇을)

- **👤 설계사** — 상담·녹음 / 갱신일·태그 입력 / 제안서·문자 검토·발송
- **🤖 AI(Claude)** — 녹취 정리·요약 / 제안서·문자 작성 (초안)
- **🔁 시스템** — 대시보드 자동 분류·집계

> 보험사 사이트 자동 로그인/수집은 하지 않습니다. 설계사 본인이 직접 확인한 정보만 정리합니다.
