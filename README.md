# 보험 영업 자동화 볼트 (간단판)

상담 녹음 → Claude로 정리 → 고객 DB → 맞춤 제안서까지, 별도 비용 없이 Claude 앱에 프롬프트를 붙여 쓰는 Obsidian 볼트입니다.

## 구성
| 파일/폴더 | 용도 |
|------|------|
| `상품DB/` | 보험 상품 정보 (한 번 정리해 재사용) |
| `고객DB/` | 고객별 가입 이력·갱신일·가족·유형 태그 |
| `상담기록/` | 상담 녹음 전사·정리 |
| `대시보드.md` | 갱신 임박·자녀보험 미가입 자동 집계 (Dataview) |
| `프롬프트.md` | 전사 정리 / 제안서 / 타깃 메시지 프롬프트 |
| `개인정보-주의.md` | 동의·보안 체크리스트 |

## 사용 순서
1. Obsidian 설치 → 이 폴더를 볼트로 열기 → **Dataview 플러그인** 설치(설정 > 커뮤니티 플러그인).
2. `상품DB`, `고객DB`의 `_..._템플릿.md`를 복사해 새 노트 작성.
3. 상담 끝나면 `프롬프트.md`의 **① 상담 정리** 프롬프트를 Claude에 붙여넣고, 결과를 `상담기록`과 `고객DB`에 반영.
4. `대시보드.md`에서 갱신 임박·자녀보험 미가입 고객 확인.
5. `프롬프트.md`의 **② 제안서**, **③ 타깃 메시지**로 마무리.

> ⚠️ 보험사 사이트 자동 로그인/수집은 하지 않습니다. 설계사 본인이 직접 확인한 정보만 정리합니다. → `개인정보-주의.md`
